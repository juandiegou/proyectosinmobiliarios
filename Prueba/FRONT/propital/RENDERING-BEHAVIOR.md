# Comportamiento de Renderizado

## React Strict Mode
Este proyecto usa React Strict Mode (`reactStrictMode: true`), que en modo desarrollo:
- ✅ Renderiza componentes dos veces intencionalmente
- ✅ Ayuda a detectar efectos secundarios
- ✅ Solo ocurre en desarrollo, NO en producción
- ℹ️ Es comportamiento esperado, no un bug
